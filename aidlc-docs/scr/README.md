# SCR (Screen Specification)

화면 명세를 관리합니다.

## 파일 구조

- `scr-001-*.md` — 화면별 명세 (순번별)

## 작성 규칙

- 모든 용어는 `/aidlc-docs/db/` 용어 사전의 엔티티명·필드명을 기준으로 통일
- 화면 ID: `SCR-001`, `SCR-002` 순번 부여
- 각 화면 명세에는 다음 항목을 포함:
  - 화면 목적
  - 진입 경로
  - UI 요소 목록 (라벨은 용어 사전 기준 용어 사용)
  - 사용자 액션 및 이벤트
  - 데이터 흐름 (localStorage 키 및 구조)
  - 유효성 검사 규칙
