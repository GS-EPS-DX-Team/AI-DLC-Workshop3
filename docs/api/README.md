# API (Backend Logic Specification)

백엔드 API 및 비즈니스 로직 명세를 관리합니다.

## 파일 구조

- `api-001-*.md` — API 엔드포인트별 명세 (순번별)

## 작성 규칙

- 모든 용어는 `/docs/db/` DB 명세의 컬럼명·테이블명을 기준으로 통일
- API ID: `API-001`, `API-002` 순번 부여
- 각 API 명세에는 다음 항목을 포함:
  - 엔드포인트 (HTTP 메서드 + 경로)
  - 요청 파라미터 / 본문 스키마
  - 응답 스키마
  - 비즈니스 로직 (처리 흐름)
  - 에러 응답
  - 관련 DB 테이블 (`/docs/db/` 참조)
  - 관련 화면 (`/docs/scr/` 참조)
